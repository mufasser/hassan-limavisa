注意事項：

申請 ESTA 的條件：

旅客護照為生物辨識電子護照且具備國民身分證號碼。 入境美國係為洽商或觀光，停留不超過90天。

透過旅行授權電子系統（ESTA）取得以VWP入境之授權許可。 以飛機或船舶入境，須為核准之運輸業者，並且有前往他國目的地之回程票。 具備足夠資金供在美期間支出。

ESTA 有效期：

通常為兩年，或到護照到期日為止。 在有效期內可多次進出美國。

需重新申請 ESTA 的情況：

換發新護照。 更改姓名、性別或國籍。 之前回答的 ESTA 申請問題已不再正確。

重要提醒：

最近常發生已獲得 ESTA 核准被取消的案例，請務必在入境美國前再次線上確認 ESTA 狀態。 如電子郵件地址變更，請務必至 ESTA 系統修改，以便接收狀態變更通知。

不適用 ESTA 的情況：

曾於2011年3月1日以後前往古巴、伊朗、伊拉克、利比亞、北韓、索馬利亞、蘇丹、敘利亞、葉門。 具有上述國家雙重國籍者。

VWP 的特殊規定：

旅客以VWP入境後到加拿大、墨西哥或鄰近島嶼短暫旅行，可再以VWP入境美國，但停留天數須扣除之前在美國及在這些地區的停留天數。 旅客無法在美國停留超過90天，且不得改變其身分（例如變更為學生簽證）。

ESTA 系統規定：

自2022年10月1日起，所有以海、空或陸路方式入境美國的旅客均須取得 ESTA 授權許可。 陸路入境美國須填具I-94W入出境紀錄表，並支付6美元之申請費用。

豁免規定： 持有伊朗、伊拉克、北韓、蘇丹、敘利亞、古巴雙重國籍者，不適用ESTA免簽，但可透過 ESTA 申請豁免。

不適用 VWP 的情況：

欲在美國工作、求學或參加交換訪客計劃者。 需過境／空勤或航海組員簽證者。 以私家飛機或私人遊艇入境美國者。 欲在美洽商或觀光超過90日者。 曾被拒絕 ESTA 授權許可，或有其他違反移民法事項者。 建議： 在出發前確認最新的 ESTA 狀態。 保持有效的電子郵件地址以接收 ESTA 狀態通知。 確保所有提交的信息準確無誤，以避免申請被拒或入境問題。
